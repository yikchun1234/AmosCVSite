# Amos Tan - 个人简历网站 (Personal Portfolio & CV) 👨‍💻

欢迎来到我的个人简历开源仓库！这是我（Tan Yik Chun / Amos）的在线简历和作品集网站。

🌐 **在线预览 (Live Demo):** (https://amoscvreview.pages.dev)

## 📖 项目简介 (Description)
这是一个用于展示我的职业经历、专业技能以及教育背景的交互式网页。作为一名**数据工程师与软件开发者**，我希望通过这个网站直观地向访客和潜在雇主展示我的技术栈和工作成就。网站不仅支持多语言切换，还包含了一个用于休闲的互动小游戏。

## ✨ 核心功能 (Key Features)
- **双语支持 (Bilingual):** 提供无缝的中文/英文双语切换，方便不同背景的访客阅读。
- **职业经历时间轴:** 清晰呈现我在数据工程、核心数据库管理、ETL流程优化以及全栈开发方面的过往经历（包括在 SF Express 等公司的任职）。
- **技能矩阵:** 分类展示我在数据库 (SQL Server, SSIS, AWS, MySQL)、编程语言 (C#, Java, PHP) 以及 BI 和自动化工具 (Power BI, Looker Studio, Power Automate) 方面的硬技能。
- **互动彩蛋 (Puzzle Challenge):** 页面底部内置了一个可选择难度 (3x3, 4x4, 5x5) 的互动拼图游戏，增加网站的趣味性和交互体验。
- **响应式布局:** 完美适配移动端和桌面端设备的浏览体验。

## 🛠️ 技术栈 (Tech Stack)
- **前端开发:** HTML5, CSS3, JavaScript *(注: 若使用了 Vue/Tailwind 等可在此处补充)*
- **平台部署:** Cloudflare Pages
