### ✨ Core Features

* 🌐 **Bilingual Interface:** Seamlessly toggle between English (EN) and 中文 (Mandarin) on the fly without reloading the page.
* 🌓 **Dynamic Theming:** Built-in Light and Dark modes utilizing custom CSS variables, radial gradients, and smooth glassmorphism transition effects.
* 🎬 **Matrix-Style Animations:** Custom `@keyframes` typing effects, blinking carets, and scroll-triggered element fade-ins powered by the modern `IntersectionObserver` API.
* 📱 **Mobile-First & Responsive:** Fully optimized for all screen sizes, featuring a collapsible hamburger navigation menu for mobile users and CSS grid/flexbox layouts.
* 🎮 **Interactive "Boredom" Mini-Game:** Includes a fully functional sliding puzzle game built natively with HTML5 `<canvas>`. Features adjustable difficulties (3x3, 4x4, 5x5) and dynamic image fetching!
* 📄 **Downloadable CV:** Direct integration for recruiters to download a PDF version of the resume.

### 📖 Portfolio Sections

1. **Hero / About:** Quick introduction and professional overview.
2. **Experience:** Detailed timeline of Data Engineering and Software Development roles (SF Express, Sense Embroidery, MAP, etc.).
3. **Projects:** Direct links to personal open-source projects (Loot Game Radar, Mahjong Score Tracker).
4. **Skills:** Categorized technical stack (Databases, Programming, Cloud Platforms, Operating Systems).
5. **Education & Languages:** Academic background, certifications, and animated language proficiency bars.
6. **Contact:** Quick-connect links to LinkedIn, Email, and WhatsApp.

---

### 🚀 Zero-Build Setup

This portfolio is incredibly lightweight and requires no backend servers, npm packages, or build tools.

1. Clone or download this repository.
2. Open `index.html` directly in any modern web browser.
3. **Deployment:** This static site can be hosted instantly for free using GitHub Pages, Cloudflare Pages, or Vercel.
